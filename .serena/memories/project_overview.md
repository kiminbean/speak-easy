SpeakEasy is a React Native + Expo SDK 54 AAC mobile app for non-verbal users. It is offline-first, stores data on device, supports 20 UI languages, has context-aware phrase prediction, TTS, caregiver alerts, custom phrases/favorites/history, and a versioned UI theme system with active style v2-liquid-glass. Key runtime split: native AI mode via react-native-executorch in native/dev builds, rule-based fallback in Expo Go or unsupported environments. Main code lives under mobile/src with app/, components/, services/, stores/, constants/, i18n/, types/, and utils/.
