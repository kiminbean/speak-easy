TypeScript project using React Native function components, Zustand stores, singleton services, StyleSheet.create for styles, and path alias @/* for src imports per AGENTS.md. Avoid as any, @ts-ignore, and inline styles. Reanimated + gesture-handler are preferred for animations/interactions. Use i18n keys instead of hardcoded UI strings. Accessibility matters: WCAG 2.1 AA, touch targets 48dp+, accessibility props on interactive elements. Services should avoid direct RN module imports when it breaks tests; native modules are mocked in jest.setup.js.
